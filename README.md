# מאיר | בניית אתרים

אתר תדמית ותיק עבודות למאיר כהן — בניית אתרים מותאמים אישית לעסקים, עמותות ואנשי ציבור.

## פיתוח מקומי

האתר סטטי לחלוטין (HTML/CSS/JS). פתיחה מקומית:

```bash
open index.html
# או שרת מקומי פשוט
python3 -m http.server 8000
```

## אירוח

האתר מתארח ב-GitHub Pages. הענף `main` מתפרסם אוטומטית.
דומיין סופי: `meirweb.co.il` — יש לחבר את הדומיין אצל הרשם (רשומות A/CNAME אל GitHub Pages) ולוודא שה-CNAME כאן תואם.

## מבנה

- `index.html` — עמוד יחיד עם כל הסעיפים
- `styles.css` — עיצוב, תמיכה בעברית ורספונסיביות
- `script.js` — תפריט נייד, שנה בפוטר
- `assets/` — פאביקון ונכסים נוספים
- `CNAME` — מיפוי הדומיין הראשי
